!DOCTYPE:

Detta är inte ett HTML-element i sig, utan det används för att deklarera vilken version av HTML som används i dokumentet. Det hjälper webbläsaren att tolka och rendera sidan korrekt.

html:

Detta element är rot- eller överordnat för hela HTML-dokumentet och omsluter allt annat innehåll på sidan.

head:

head-elementet innehåller metadata om dokumentet, som titel, teckenkodning och länkar till externa resurser. Det är osynligt för användaren och påverkar inte den visuella presentationen av sidan.

title:

title-elementet används för att ange sidans titel, som visas i webbläsarens titelrad eller i bokmärken. Det hjälper också till med sökmotoroptimering.

style:

style-elementet används för att definiera inlinestilregler för att styra sidans utseende, inklusive färg, typsnitt, layout och mycket mer.

link:

link-elementet används främst för att länka till externa stilmallar (CSS) och ikoner, som favicon. Det gör det möjligt att separera stilinformation från innehållet i HTML-dokumentet.

body:

body-elementet omsluter den faktiska innehållet som ska visas på webbsidan, inklusive text, bilder, länkar och andra element.

p:

p-elementet representerar en avsnittsvis text och skapar en ny rad efter avsnittet. Det används ofta för att formatera och separera textinnehållet.

Headings (h1 till h6):

Dessa är rubrikselement som används för att strukturera sidan och ge viktiga rubriker och underrubriker. h1 är den högsta nivån, medan h6 är den lägsta.

ol:

ol står för ordnad lista och används för att skapa en lista där varje objekt är numrerat. Varje objekt omsluts av ett li-element.

ul:

ul står för oordnad lista och används för att skapa en lista där objekten inte är numrerade, vanligtvis med punktmarkörer. Varje objekt omsluts av ett li-element.

li:

li står för listobjekt och används inuti ol eller ul för att definiera enskilda objekt i en lista.
